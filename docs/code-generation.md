## What You Can Generate

The toolkit can generate boilerplate code from your event models, including:

- Event definitions and schemas
- Command handlers and aggregates
- Event handlers and projections
- API endpoints (REST, GraphQL)
- Database schemas and migrations
- Test scaffolding and fixtures
- Type definitions and interfaces

## Supported Languages and Frameworks

### Backend Frameworks

- **Java**: Spring Boot, Axon Framework, Eventuate
- **C#/.NET**: ASP.NET Core, MassTransit, NServiceBus
- **Node.js**: Express, NestJS, EventStore
- **Python**: FastAPI, Django, Flask
- **Go**: Echo, Gin, NATS
- **Kotlin**: Spring Boot, Ktor

### Frontend Frameworks

- **React**: TypeScript, Redux, React Query
- **Vue.js**: TypeScript, Pinia, Composition API
- **Angular**: TypeScript, NgRx, RxJS

## Benefits

- **Consistency**: Standardized code across your team
- **Speed**: Generate boilerplate in seconds, not hours
- **Accuracy**: Code perfectly matches your event model
- **Maintainability**: Regenerate when your model changes
- **Best Practices**: Templates follow framework conventions
