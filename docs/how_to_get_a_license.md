# How to get a License

You can use the Miro Event Modeling Toolkit for free in the Community Edition.
This already allows to you take advantage of Modeling in a structured way.

In the commercial Edition, you´ll find much more benefits like:

✅ Live AI validation

✅ Custom code generators

✅ Export to EventCatalog

✅ Axon Framework generator

✅ Live in-browser prototype

✅ GDPR data path analysis

✅ System modernization planning

✅ Custom Screen Drawing

## How to get a license

You can simply purchase a License [here](https://copecart.com/products/a8a9c0b1/checkout)

You´ll then receive an E-Mail with instructions how to activate the license.

Just copy your License Id and send by Mail to license@nebulit.de

If the email differs from the E-Mail you ordered, make sure to include the PDF or a hint to the Order you placed, 
so we can check the license.

![User ID]({{ '/assets/images/license001.png' | relative_url }})

You License is activated either immediately or within 1-2 hours typically.

## Cancelling a license

You can cancel your license manually at any point in time. There is a link in your order confirmation E-Mail showing how to do that.
