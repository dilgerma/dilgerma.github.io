# Drawing

Using the Screen Drawer allows you to quickly draw screens using a simple Drawing Tool.

![Screen Drawer]({{ '/assets/images/tutorial/elements_view/drawing/draw001.png' | relative_url }})

The tool used here is [TLDRAW](https://www.tldraw.com/).

My rule of thumb is simple - never use more than 2 minutes to draw a screen.

An exported Screen can be re-edited - ** but will always result in a new screen **

So an existing screen cannot be changed anymore.
